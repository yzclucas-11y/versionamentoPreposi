Aula de Versionamento

  1. Crie um repositório no GitHub com o mesmo nome do seu projeto.
  2. No terminal, inicialize o repositório local com o comando: git init.
  3. Altere o nome da branch principal para main.
  4. Adicione os arquivos com o comando: git add . e faça o commit com git commit -m "first commit".
  5. Conecte ao repositório remoto com o comando: 
    git remote add origin https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
  6. Envie os arquivos com o comando: git push -u origin main.
  7. No GitHub, vá em Settings > Pages.
  8. Em "Branch", selecione main e a pasta / (root). Clique em Save.
  9. Após alguns segundos, seu site estará disponível no link exibido em Pages.
